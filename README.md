# Descripción del proyecto

Este proyecto consiste en el desarrollo de un **formulario de registro interactivo** utilizando JavaScript, el cual implementa validación de datos con retroalimentación visual en tiempo real.

El formulario permite al usuario ingresar sus datos y valida automáticamente cada campo, proporcionando mensajes personalizados y mejorando la experiencia de usuario.

El objetivo principal es aplicar conceptos fundamentales de JavaScript como:

- Manipulación del DOM
- Manejo de eventos
- Validación de formularios
- Uso de la Constraint Validation API
- Retroalimentación visual dinámica
